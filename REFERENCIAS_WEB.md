### Nameservers (DNS Owl — opción recomendada)

Si mantienes DNS administrado en DNS Owl, los nameservers deben ser:

- `ns1.dnsowl.com`
- `ns2.dnsowl.com`
- `ns3.dnsowl.com`

**Problema detectado:** el dominio apuntaba a `ns1.facilweb.info` / `ns2.facilweb.info` (IP `162.222.227.178`), que rechazaba consultas DNS. Los registros en DNS Owl no aplican hasta corregir los nameservers en el registrador.

## Enfoque adoptado para Informática González

Híbrido **70% visual / 30% informativo** — no copiar agencia de moda al 100%, sino combinar:

| De agencias premium | Mantener propio |
|---------------------|-----------------|
| Nav corto (3 links + CTA) | WhatsApp flotante (LATAM) |
| Works antes que servicios | Stack técnico visible en proyectos |
| Hero + statement con scroll | Formulario / CTA de contacto B2B |
| Marquee + tipografía grande | Casos reales con tech stack |
| Acordeón de servicios (923) | CRM como sección aparte |

       → CrmIntegrations → ToolsWeUse → About → FooterCTA → Footer
